# TradingAgents 视频文案优化设计

> 日期: 2026-04-12
> 状态: 待审阅
> 优化对象: `docs/tradingagents-video-copy.md` 中的 7 段配音文案 + 画面方向

## 优化目标

在保持"诚实深度"路线的前提下，优化配音文案的节奏、金句密度和口语化程度，同时调整画面方向以适应竖屏 1080×1920 的展示约束。

## 确认的设计决策

| 决策项 | 选择 | 理由 |
|--------|------|------|
| 整体路线 | A: 保持"诚实深度" | 场景 5 的冷水场景是核心差异化 |
| 口播风格 | 关键场景加情绪张力 | Hook/辩论更有力，数据/CTA 保持克制 |
| 总时长 | ~150s（与原 ~154s 接近） | 金融交易题材信息量大，时长合理 |
| 图片使用 | React UI 为主 + 图片作证据闪现 | 避免竖屏下截图字小看不清 |

## 情绪曲线

```
场景1(Hook)  → 🔥 热：短句冲突，制造紧张感
场景2(定位)  → 📐 稳：沉稳类比，建立信任
场景3(辩论)  → 🔥 热：有力对比，制造戏剧性
场景4(风控)  → 📐 稳：结构叙事，展现严谨
场景5(数据)  → ❄️ 冷：先扬后抑，泼冷水
场景6(生态)  → 📐 稳：实用导向，给行动路径
场景7(CTA)   → 🎯 收：克制判断，金句收束
```

## 配音文案优化（逐场景）

### 场景 1: Hook — "技术说买、财报说卖、推特说崩"

**时长**: ~17s

**优化后文案**:

> 技术指标说强烈买入...财报数据说谨慎观望...推特上全在喊崩...你到底该听谁的...GitHub 上有个项目...四万九千颗星...它的解法不是让一个 AI 给你答案...而是让一整个 AI 团队...先吵一架...再给你结论

**改动摘要**:
- 删掉中间"做交易...最难的不是选股...是面对互相矛盾的信息..."的转折铺垫
- 直接用"你到底该听谁的"一句收束三冲突
- "思路" → "解法"：更有 solution 感

---

### 场景 2: 定位 — "模拟一个投行交易部门"

**时长**: ~22s

**优化后文案**:

> TradingAgents 来自 UCLA 和 MIT 研究者的论文...核心思路就一句话...你去看任何一个投行的交易部门...不可能是一个人拍脑袋做决策...而是一群人各管一摊...分析师负责看数据...研究员负责吵架...风控负责踩刹车...最后由老板拍板...TradingAgents 就是把这套组织架构...原封不动搬到了 AI 上...九个角色...一条完整的决策链

**改动摘要**:
- 删除重复的"报菜名"（原文同一信息讲了两遍）
- 用"看数据/吵架/踩刹车"三句概括代替逐一列举九个角色
- "核心思路很直接" → "核心思路就一句话"：更有自信感

---

### 场景 3: 核心亮点 — "多空辩论：让 AI 先吵起来"

**时长**: ~22s

**优化后文案**:

> 这是整个框架最有意思的设计...四份分析报告交上来之后...不是直接汇总给结论...而是先让两个研究员...一个站多头...一个站空头...进行三轮辩论...看多的会找一切利好证据反驳...看空的会死磕每一个风险不放...观点越对立...盲点暴露得越彻底...你想想...一个 AI 给你的是一个观点...一群 AI 吵完架给你的...是一个经过正反压力测试的判断

**改动摘要**:
- 删除尾部全部回测数据（AAPL +26%、B&H -5%、最大回撤），移到场景 5 独占
- 加入"三轮辩论"具体细节
- 尾部新增金句："一个 AI 给你的是一个观点...一群 AI 吵完架给你的...是一个经过正反压力测试的判断"

---

### 场景 4: 深入 1 — "全流程风控，不是摆设"

**时长**: ~20s

**优化后文案**:

> 辩论出结论后...交易员先拿出一套完整方案...但还不能下单...要过风控这一关...波动率、流动性、持仓集中度...逐项过清单...不达标直接打回...而且风控不是一个人...是三个...一个激进、一个中性、一个保守...三种风格的风控经理也要先吵一架...最后由投资组合经理综合拍板...这意味着...在这个系统里...连风控审批都有对抗机制

**改动摘要**:
- 删掉"和真实投行的交易台几乎一样"（场景 2 已做类比）
- 三种风控策略"也辩论"从末尾补充位提到中段核心位置
- 尾部新增金句："连风控审批都有对抗机制"

---

### 场景 5: 深入 2 — "论文数据亮眼，但得看清前提"

**时长**: ~24s

**优化后文案**:

> 该聊数据了...论文回测了苹果、谷歌和亚马逊...三只股票三个月...结果相当亮眼...苹果累积回报百分之二十六...同期买入持有是负百分之五...看到这个数字...你可能已经心动了...但先别急...同样的代码同样的参数...社区有人复现苹果...跑出来是负百分之二十五...正负五十个百分点的差距...出在同一个框架上...论文自己也承认...三个月回测太短...Sharpe 高达八明显偏高...所以这个数据...能参考...但远不到信仰

**改动摘要**:
- "说数据" → "该聊数据了"：更有"正式开话题"感
- 只详细讲 AAPL 一只，不逐一念三只报表
- "你可能已经心动了...但先别急..."——先扬后抑的戏剧化反转
- 社区复现紧跟论文数据，形成直接对比（+26% vs -25%）
- "别当圣经" → "远不到信仰"：更有分量

---

### 场景 6: 深入 3 — "开源生态 + 中文版支持 A 股"

**时长**: ~22s

**优化后文案**:

> 好消息是...全部开源...Apache 二点零...GPT Claude Gemini Grok 随便选...也能用 Ollama 跑本地不花钱...Docker 一键起来就能用...而且...有人做了一个中文增强版...两万颗星...直接支持 A 股和港股...已经生成了一百五十多份 A 股研报...你不需要写一行代码...Web 界面打开就能看到完整分析...对中国投资者来说...这个中文版可能比原版更实用

**改动摘要**:
- 砍掉 Trading-R1（不是当前框架功能，分散焦点）
- 砍掉 Tushare/AkShare（技术细节，普通观众不关心）
- 信息点从 10+ 压缩到 6 个
- "也能用 Ollama 跑本地模型" → "也能用 Ollama 跑本地不花钱"：点出成本
- 尾部新增判断句："对中国投资者来说...这个中文版可能比原版更实用"

---

### 场景 7: CTA — "AI 不能替你赚钱，但能替你想清楚"

**时长**: ~20s

**优化后文案**:

> 所以...这个框架值不值得看...我的判断是...值得研究...但不值得盲信...它不能帮你赚钱...但它能帮你在一堆互相矛盾的信息里...做出更有逻辑的判断...如果你也觉得一个人盯盘加一个 AI...越来越不够用了...这套让 AI 先吵架再给结论的思路...值得花时间了解一下...A 股用户直接看中文版...链接放评论区...投资有风险...DYOR

**改动摘要**:
- 删除重复冷水，替换为一句收束："值得研究...但不值得盲信"
- "这套多角色协作加对抗辩论的思路" → "这套让 AI 先吵架再给结论的思路"：回扣 Hook 金句
- "投资有风险...DYOR" 比单独 "DYOR" 更完整合规
- 从 22s 压缩到 ~20s

## 画面方向优化（逐场景）

### 场景 1: Hook

- 三色冲突卡片改为**竖排 stagger**（竖屏更友好）
- 删掉底部金句文字（配音已说，画面不重复）
- Stars 计数器 `★ 49,000+` 放大，成为视觉焦点
- `TauricResearch.png` logo 缩小，右上角品牌水印

### 场景 2: 定位

- **主体改为 React 垂直决策管线**（从上到下流动）：
  - 顶层：4 分析师图标（📊💬📰📈），stagger 入场
  - 中层：多空辩论 ⚔️ 标记
  - 底层：交易 → 风控 → 审批 → 执行，lineGrow 动画
- `schema.png` 降级为 opacity: 0.08 + blur(4px) 背景纹理
- 右上角标签：`arXiv:2412.20138 · UCLA × MIT`
- 不再使用 `cli_news.png`

### 场景 3: 辩论

- **删掉底部数据条**（数据移到场景 5）
- `analyst.png` 改为缩放聚焦单个分析师卡片（3 秒暗示"四份报告"）
- `researcher.png` 保持不变（Bullish ↔ Bearish 对比图）
- 辩论过程中加 VS 火花动画（pulseGlow on ⚔️）
- 配音说"三轮辩论"时，画面顶部出现 `Round 1→2→3` 进度条

### 场景 4: 风控

- **主体改为 React 垂直管线**（3 层）：
  - 交易员方案节点 → lineGrow → 三路风控（激进🔴/中性🟡/保守🟢）→ lineGrow → 组合经理审批
- `risk.png` 以缩放聚焦方式闪现 Risky/Neutral/Safe 三栏（4 秒后 fadeOut）
- `trader.png` 不使用
- 配音说"连风控审批都有对抗机制"时，三路风控之间出现 ⚔️ 碰撞动画

### 场景 5: 数据两面

- **第一阶段（数据亮眼时）**：全屏金色/绿色面板
  - 中央大字 `AAPL +26.62%`（金色 monospace 90px，numberCountUp）
  - 对比小字 `Buy & Hold: -5.23%`（红色）
  - 成功绿色 radial-gradient 光晕
- **第二阶段（"先别急"时）**：色调翻转 绿 → 红
  - 底部升起红色面板：`AAPL -25%`（社区复现）
  - 分割线 + ⚠️ 标记
  - 底部标签：`仅 3 个月 · 仅 3 只股票 · Sharpe 偏高`
- 不使用仓库图片，纯数据可视化

### 场景 6: 生态

- **主体：双项目对比卡片**（竖排）
  - 上卡：`TradingAgents ★ 49,000+`（蓝色边框）+ 标签
  - 下卡：`TradingAgents-CN ★ 20,800+`（红金边框 🇨🇳）+ A 股/港股/158+ 报告
- `cli_init.png` 小尺寸闪现（配音说"Docker 一键"时，约 3 秒）
- 删掉 `cli_technical.png` 和 Trading-R1 路线图
- 配音说"中文增强版"时，下卡 slideUp 入场并放大

### 场景 7: CTA

- **删掉两列对比面板**（场景 5 已做"数据两面"）
- 上方：`值得研究 ✅` 和 `不值得盲信 ⚠️` 左右 slideIn
- 中间大字金句（渐变色）：**"AI 不能替你赚钱，但能替你想清楚"**
- 下方：传送门链接（GitHub 两项目 + 论文）
- `cli_transaction.png` 缩放聚焦闪现 3 秒（展示"完整决策输出长什么样"）
- 底部：⚠️ 仅供研究，不构成投资建议
