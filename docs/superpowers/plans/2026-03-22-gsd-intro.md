# GSD Intro Video Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a 7-scene vertical short video (1080x1920) introducing GSD (Get Shit Done) framework, following the pain-point driven narrative with green terminal visual style.

**Architecture:** Remotion composition with Zod schema, 7 scene components, shared animation library, TTS voiceover pipeline, and VTT-based subtitle sync. Follows existing vertical video patterns (SuperPowers, OpenClawAI) with green terminal color scheme.

**Tech Stack:** Remotion, React, TypeScript, Zod, edge-tts, music-metadata

**Spec:** `docs/superpowers/specs/2026-03-22-gsd-intro-design.md`
